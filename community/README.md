# Community

Forked, adapted, or linked plugins from the broader ecosystem.

Each subdirectory must contain the same required files as first-party plugins (`README.md`, `LICENSE`, `.claude-plugin/plugin.json`), plus attribution and a link back to the upstream source in the plugin's `README.md` and `plugin.json` `repository` field.

See the [root README](../README.md) for the full directory structure and conventions.

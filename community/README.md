# community/

Third-party plugins that have been forked, adapted, or linked from external sources.

Same requirements as first-party plugins (`README.md`, `LICENSE`, `.claude-plugin/plugin.json`), plus:

- Attribution to the original author in `README.md`
- A `repository` field in `plugin.json` pointing to the upstream source
- Clear documentation of any modifications from the original

See the [root README](../README.md) for the full directory layout and conventions.
